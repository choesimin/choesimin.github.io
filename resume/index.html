남을 이롭게 하며 사는 삶이 저에겐 가치있는 것이고, 그래서 사람과 가까운 곳에서 이로움을 주며 살아가고 싶었습니다. 그러던 중 개발을 접하게 되었습니다. 사람 가까이에서 이로움을 전할 수 있는 분야라는 생각과 개발로써 무언가를 만들어낼 때 생기는 즐거움에 빠져 본격적으로 개발 공부를 시작하게 되었습니다.

tech skills

저는 지금 이것을 할 수 있습니다.

저의 첫 개발 공부는 자율주행 대회를 위한 공부였습니다. Ubuntu에 ROS(Robot Operating System)를 설치하고, Github의 오픈소스 라이브러리를 이용하여 자율주행 시뮬레이션을 구현했습니다. 또 이때, C++를 통해 프로그래밍 언어를 처음 접하면서 기본적인 문법을 익힐 수 있었습니다.
웹개발자가 되기로 결정하고서는 생활코딩에서 HTML, CSS, JavaScript 강의를 들으며 기본적인 화면 구성, 반응형 페이지 구현 등을 학습했습니다. 방학 때는 IT Korea Academy에서 Java와 Python 과정을 들었고, 현재는 같은 교육기관의 Java Backend 개발자 과정 수강 중에 있습니다. Spring Framework으로 서버를 구성하여 페이징 기능이 있는 게시판, 로그인 등의 웹서비스를 개발할 수 있습니다. Node.js를 학습하여 소규모 웹서비스에 적합한 Web Server를 만들 수 있습니다.

진행한 프로젝트는 Amazon의 EC2, ELB, Route 53, Certificate Manager를 사용하여 https로 배포하였고, Tomcat, MySql로 관리하고 있습니다.
또한 Git을 사용하여 프로젝트의 소스 코드를 관리하고 있으며, 이해를 돕기 위해 README를 작성하여 대략적인 개발의 흐름을 나타내었습니다.
