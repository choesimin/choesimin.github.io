## Database를 조작하는 여러 SQL 명령어

- SQL 명령어는 database를 관리하고 조작하기 위한 표준화된 명령어 집합입니다.

- SQL 명령어는 각각의 역할과 특성에 따라 DDL, DML, DCL, TCL로 분류됩니다.
    - **DDL**은 database 구조를 정의합니다.
    - **DML**은 data를 조작합니다.
    - **DCL**은 접근 권한을 제어합니다.
    - **TCL**은 transaction을 관리합니다.
